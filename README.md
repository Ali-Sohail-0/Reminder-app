# Reminder-app

This is a simple reminder app made in Python.

It lets you:

Set a time after which reminder will trigger

Repeat the reminder multiple times

Add snooze time between reminders

Hear reminders using Windows voice (SAPI)

How to run

Run the Python file:

python reminder app.py

What I practiced

functions

loops

input handling

basic error handling

using time module

using win32com for voice

Note

This is a beginner project made by me while learning Python.
